# vue-i18n-manager
Multi-language manager for Vue
>**Vue 2.0 and Vuex are required**

### Documentation
Docs are available [here](https://matteogabriele.gitbooks.io/vue-i18n-manager/content/)

### Install
```bash
$ npm install vue-i18n-manager
```
